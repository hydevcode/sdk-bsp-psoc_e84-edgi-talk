◆ cy_stc_autanalog_stt_prb_t

struct cy_stc_autanalog_stt_prb_t
Data Fields
bool unlock PRB Unlock:
FALSE - Locked:
Data from fields prbVrefxFw and prbVrefxTap will NOT be taken into account in the corresponding state of STT
TRUE - Unlocked:
Data from fields prbVrefxFw and prbVrefxTap will be used in the STT corresponding state.
bool prbVref0Fw PRB Vref0.

The control of PRB Vref0 from the firmware, Vref0 must be enabled in cy_stc_autanalog_prb_cfg_t
FALSE - the PRB output determined by the field prbVref0Tap in this structure
TRUE - the PRB output determined by the tap field, defined in cy_stc_autanalog_prb_cfg_t

cy_en_autanalog_prb_tap_t prbVref0Tap The tap selector for the required Vref0.
bool prbVref1Fw PRB Vref1.

The control of PRB Vref0 from the firmware, Vref1 must be enabled in cy_stc_autanalog_prb_cfg_t
FALSE - the PRB output determined by the field prbVref1Tap in this structure
TRUE - the PRB output determined by the tap field, defined in cy_stc_autanalog_prb_cfg_t

cy_en_autanalog_prb_tap_t prbVref1Tap The tap selector for the required Vref1.
